연수원 여기어때

서비스 소개

전국의 기업 연수원 및 주요 교육 시설 정보를 지도 위에서 한눈에 확인할 수 있는 서비스입니다. 현재 기업 연수원 기준 350여 개 이상의 방대한 시설 정보를 제공합니다.

이용 방법

  • 지도 클릭 — 마커를 클릭하면 연수원 이름, 주소, 홈페이지·네이버 지도 링크를 확인할 수 있습니다.
  • 검색 — 상단 검색창에 연수원 이름을 입력하면 자동완성 결과가 표시됩니다.
  • 연수원 목록 — 지역별 검색, 카드/테이블 보기, 페이지 탐색이 가능합니다.
  • 클러스터 — 지도를 축소하면 가까운 연수원이 묶여 표시되고, 확대하면 개별 마커로 분리됩니다.

데이터 출처

연수원 정보는 네이버 지역 검색 API를 통해 수집되었으며, 주기적으로 업데이트됩니다.
일부 정보는 실제와 다를 수 있으니 방문 전 반드시 공식 홈페이지 또는 전화로 확인하시기 바랍니다.

네이버 지역 검색 API 네이버 지도 API

유의사항

  • 연수원 정보(운영 여부, 요금 등)는 변경될 수 있습니다.
  • 지도 마커 위치는 주소 기반으로 표시되어 실제 위치와 차이가 있을 수 있습니다.
  • 누락된 연수원이나 오류 정보는 관리자에게 문의해 주세요.
